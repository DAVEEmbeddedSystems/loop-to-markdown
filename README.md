# Loop to Markdown

A Tampermonkey userscript that converts Microsoft Loop pages to Markdown with one click.

## Features

- Tables, code blocks, headings, lists (ordered, unordered, checkboxes)
- @mentions, hyperlinks, inline code
- Bold text detection
- Code language auto-detection (Python, JavaScript, SQL, YAML, JSON, Mermaid, etc.)
- Quip link capture

## Installation

1. Install [Tampermonkey](https://www.tampermonkey.net/) browser extension
2. Click [here to install the script](https://raw.githubusercontent.com/oztalha/loop-to-markdown/main/loop-to-markdown.user.js)

## Usage

1. Open any Microsoft Loop page
2. Click the "📋 Copy as Markdown" button (bottom-left)
3. Paste the markdown anywhere
